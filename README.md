# Full Stack Playground 全栈试验场

Full Stack Playground (fspg) 是一个用于个人全栈技能修炼的代码片段试验场。**该项目不不是完整呈现完整的项目体系，而是专注于技能点的学习和记录**。

## 技术栈

本项目涉及以下技术：

- HTML
- CSS
- JavaScript
- Bootstrap
- React
- UI/UX
- Node.js
- Express
- Git
- GitHub
- 软件设计

## 安装步骤

1. 克隆该项目到本地：
   ```bash
   git clone https://github.com/practicenotes007/fspg.git
   ```
2. 进入项目目录：
   ```bash
   cd fspg
   ```
3. 安装依赖：
   ```bash
   npm install
   ```

## 使用方法

1. 启动开发服务器：
   ```bash
   npm start
   ```
2. 打开浏览器，访问 `http://localhost:3000` 查看项目。
